# make-transimg

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

A tool to create transparent images and copy them to the clipboard.

## Demo
https://code4fukui.github.io/make-transimg/

## Features
- Create transparent images of custom size
- Copy the created image to the clipboard

## Usage
1. Enter the desired width and height in the input fields.
2. Click the "make & copy" button to generate a transparent image and copy it to the clipboard.
