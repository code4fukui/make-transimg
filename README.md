# make-transimg

A simple web tool to generate a fully transparent PNG image of any size and copy it directly to your clipboard.

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

## Demo

**https://github.com/code4fukui/make-transimg

The user interface is minimal, consisting of two input fields for width and height and a single "make & copy" button.

## Features

-   **Custom Dimensions:** Create a transparent PNG of any specified width and height.
-   **One-Click Copy:** Copies the generated image to the clipboard instantly.
-   **Client-Side:** Works entirely in your browser using the Canvas and Clipboard APIs. No data is uploaded.

## Usage

1.  Open the [demo page](https://code4fukui.github.io/make-transimg/).
2.  Enter the desired `width` and `height` in pixels.
3.  Click the **make & copy** button.
4.  The transparent image is now on your clipboard, ready to be pasted into other applications like design software, presentation slides, or documents.

## Use Cases

-   Creating spacer images for web layouts.
-   Pasting an invisible placeholder into design tools (Figma, Photoshop) or presentation software (PowerPoint, Google Slides).
-   Quickly generating a blank canvas of a specific size for editing.
