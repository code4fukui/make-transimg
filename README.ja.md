# make-transimg

任意のサイズの完全な透過PNG画像を生成し、クリップボードに直接コピーできるシンプルなWebツールです。

## デモ

**https://code4fukui.github.io/make-transimg/**

ユーザーインターフェースは最小限で、幅と高さの2つの入力フィールドと、「make & copy」ボタン1つのみで構成されています。

## 機能

- **カスタムサイズ:** 指定した幅と高さの透過PNGを作成できます。
- **ワンクリックコピー:** 生成された画像を即座にクリップボードへコピーします。
- **クライアントサイド:** Canvas APIとClipboard APIを使用し、すべてブラウザ上で動作します。データがアップロードされることはありません。

## 使い方

1. [デモページ](https://code4fukui.github.io/make-transimg/)を開きます。
2. 希望する `width`（幅）と `height`（高さ）をピクセル単位で入力します。
3. **make & copy** ボタンをクリックします。
4. 透過画像がクリップボードにコピーされ、デザインソフト、プレゼンテーションスライド、ドキュメントなどの他のアプリケーションに貼り付けられる状態になります。

## 使用例

- Webレイアウト用のスペーサー画像の作成。
- デザインツール（Figma、Photoshop）やプレゼンテーションソフト（PowerPoint、Google Slides）への透明なプレースホルダーの貼り付け。
- 編集用の特定サイズの空白キャンバスの迅速な生成。
